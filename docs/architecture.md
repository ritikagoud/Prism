# Prism Architecture

## Overview

Prism is a multi-agent due diligence platform that analyzes startup pitches, validates claims, identifies risks, researches competitors, and generates evidence-backed investment insights.

## User Flow

User Uploads Pitch Deck / Startup Description

↓

Orchestrator Agent

↓

Claim Extraction Agent

↓

Research Agent

↓

Competitor Agent

↓

Risk Assessment Agent

↓

Verifier Agent

↓

Report Generation Agent

↓

Final Due Diligence Report

## Agent Roles

### Orchestrator Agent

Coordinates workflow and delegates tasks to specialized agents.

### Claim Extraction Agent

Extracts business, market, financial, and product claims.

### Research Agent

Finds supporting and contradicting evidence.

### Competitor Agent

Identifies direct and indirect competitors.

### Risk Assessment Agent

Evaluates business, market, execution, and technology risks.

### Verifier Agent

Reviews outputs and validates confidence levels.

### Report Generation Agent

Produces a structured executive summary and final report.
